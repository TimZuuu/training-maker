# Log bezpośrednich poprawek od użytkownika

1. Niezgodność nazw: jeśli mowa o tabeli sprzedaży, trzymać spójność i nie mieszać nazw.
- Status: wdrożone przez użycie nazw ustalonych przez usera / placeholderów.

2. Brak `TEXT` przed taskami odniesień tabelarycznych (`#`).
- Status: wdrożone w ETAP 1A (`TEXT` przed TASK 1 i TASK 2/3).

3. `#Totals` to nie sam `FILL`; powinien zawierać akcję użytkownika.
- Status: wdrożone (`TASK — Włączenie Total Row` + osobny `TEXT` wyjaśniający).

4. `@` poza tabelą: formuła ma zawierać nazwę tabeli i zadanie przeciągania poza tabelę.
- Status: wdrożone (`=[nazwa_tabeli][@Ilość]` + przeciąganie).

5. Nie podawać użytkownikowi gotowej nazwy faktu sprzedaży jako przykładu rozstrzygającego.
- Status: wdrożone (user sam klasyfikuje i nazywa sprzedaż).
